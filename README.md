# Portfolio | amaulana8

Cinematic single-page portfolio built with **Vite + React + TypeScript + Tailwind CSS + Framer Motion**.

## Stack
- Vite 5 + React 18
- TypeScript
- Tailwind CSS
- Framer Motion (animasi)
- Lenis (smooth scroll)

## Sections
- Hero
- About (timeline karir)
- Projects
- Skills & Expertise
- Contact (social links)

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Push ke branch `main`, GitHub Actions otomatis deploy ke GitHub Pages.
